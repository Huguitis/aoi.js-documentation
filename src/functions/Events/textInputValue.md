`$textInputValue` returns data of a context menu option.

## Usage

```php
$textInputValue[option]
```

## Parameters

| Field  | Type   | Description               | Required |
|--------|--------|---------------------------|:--------:|
| option | string | option <br /> 1. **name** |   true   |
