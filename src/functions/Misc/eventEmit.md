`$eventEmit` will emit a custom event.

## Usage

```php
$eventEmit[name;...datas]
```

## Parameters

| Field     | Type     | Description                                                        | Required |
|-----------|----------|--------------------------------------------------------------------|:--------:|
| name    | string   | custom event name                                                         |   true   |
| ...datas    | string   | custom event data     |   true   |
