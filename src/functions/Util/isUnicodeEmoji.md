`$isUnicodeEmoji` will check if the given emoji is an unicode emoji.

## Usage

```php
$isUnicodeEmoji[emoji]
```

## Parameters

| Field | Type   | Description   | Required |
|-------|--------|---------------|----------|
| emoji | string | unicode emoji | true     |

## Example

This will return `true` as "ðŸ¤“" is an valid unicode emoji:

```javascript
bot.command({
    name: 'isUnicodeEmoji',
    code: `
  $isUnicodeEmoji[ðŸ¤“]
  `
});
```
