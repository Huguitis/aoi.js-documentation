`$oldChannel` holds data of the channel before it was updated, used in channel update callback, or else any data will be
empty

## Usage

```php
$oldChannel[option]
```

## Parameters

| Field  | Type   | Description               | Required |
|--------|--------|---------------------------|:--------:|
| option | string | option <br /> 1. **name** |   true   |
