`$timeoutData` holds data for $setTimeout data parameters, this will be empty if used outside timeoutCommand codes.

## Usage

```php
$timeoutData[option]
```

## Parameters

| Field  | Type   | Description | Required |
|--------|--------|-------------|:--------:|
| option | string | option name |   true   |
