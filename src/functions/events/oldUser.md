`$oldUser` holds data of the old user, used in update user callback, or else any data will be empty.

## Usage

```php
$oldUser[option]
```

## Parameters 


| Field     | Type    | Description                                        | Required |
|-----------|---------|----------------------------------------------------| :------: |
| option    | string  | option <br> 1. **name**                            | yes      |
