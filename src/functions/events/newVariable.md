`$newVariable` gets the data from variable callbacks.

## Usage

```php
$newVariable[option;sep?]
```

## Parameters 


| Field     | Type    | Description                                        | Required |
|-----------|---------|----------------------------------------------------| :------: |
| option    | string  | option <br /> 1. **name**                            | yes      |
| sep    | string  | seperator                           | no      |
