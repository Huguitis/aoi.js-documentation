`$fetchClientValues` will return data of all client shards.

## Usage

```php
$fetchClientValues[func]
```

## Parameters

| Field | Type   | Description         | Required |
| ----- | ------ | ------------------- |:--------:|
| func  | string | function to execute |    true   |
