`$spliceTextJoin`

## Usage

```php
$spliceTextJoin[text;splitter;sep1?;sep2?]
```

## Parameters

| Field    | Type   | Description | Required |
|----------|--------|-------------|----------|
| text     | string |             | true     |
| splitter | string |             | true     |
| sep1     | string |             | false    |
| sep2     | string |             | false    |
