`$multi` will multiplicate given numbers.

## Usage

```php
$multi[...numbers;...numbers]
```

## Parameters

| Field   | Type   | Description                  | Required |
|---------|--------|------------------------------|----------|
| numbers | number | numbers you want to multiply | true     |

## Example

This will return `72` as `8*9` equals that:

```javascript
bot.command({
    name: 'multi',
    code: `
  $multi[8;9]
  `
});
```
