`$newChannel` holds data of the updated channel, used in channel update, delete and create callbacks, or else any data
will be empty.

## Usage

```php
$newChannel[option]
```

## Parameters

| Field  | Type   | Description               | Required |
|--------|--------|---------------------------|:--------:|
| option | string | option <br /> 1. **name** |   true   |
