`$awaitData` will return awaited data given in awaited commands.

## Usage

```php
$awaitData[name]
```

## Parameters

| Field     | Type     | Description                                                        | Required |
|-----------|----------|--------------------------------------------------------------------|:--------:|
| name      | string   | awaited data name                                                  |   true   |
