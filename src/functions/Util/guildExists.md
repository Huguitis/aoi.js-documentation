`$guildExists` will check if the given guild exists.

## Usage

```php
$guildExists[guildId]
```

## Parameters

| Field   | Type    | Description | Required |
|---------|---------|-------------|----------|
| guildId | integer | guild id    | true     |

## Example

This will return `true` your guild exists:

```javascript
bot.command({
    name: 'guildExists',
    code: `
  $guildExists[$guildID]
  `
});
```
