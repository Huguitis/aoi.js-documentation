`$interactionData` will return given arguments of an interaction.

## Usage

```php
$interactionData[data]
```

## Parameters 


| Field     | Type    | Description                                        | Required |
|-----------|---------|----------------------------------------------------| :------: |
| data    | integer  | data to retrieve                             | yes      |
