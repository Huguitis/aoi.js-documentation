`$oldPresence` holds old data of the user's presence, used in update presence callback, or else any data will be empty

## Usage

```php
$oldPresence[option]
```

## Parameters

| Field  | Type   | Description               | Required |
|--------|--------|---------------------------|:--------:|
| option | string | option <br /> 1. **name** |   true   |
