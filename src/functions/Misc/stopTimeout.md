---
title: $stopTimeout
description: $stopTimeout will end/stop a timeout created by $setTimeout.
id: stopTimeout
---

`$stopTimeout` will stop a timeout created by `$setTimeout`.

## Usage

```php
$stopTimeout[id]
```

## Parameters

| Field     | Type     | Description                                                        | Required |
|-----------|----------|--------------------------------------------------------------------|:--------:|
| id    | string   | timeout command ID                                                         |   true   |
