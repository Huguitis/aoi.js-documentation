`$interactionDefer` defers an Interaction of the last 15 minutes.

## Usage

```php
$interactionDefer[ephemeral]
```

## Parameters

| Field     | Type    | Description                                    | Required |
|-----------|---------|------------------------------------------------|:--------:|
| ephemeral | integer | visible to the command author only? true/false |   true   |
