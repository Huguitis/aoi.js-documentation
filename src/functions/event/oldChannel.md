`$oldChannel` holds data of the channel before it was updated, used in channel update event, or else any data will be
empty

## Usage

```php
$oldChannel[option]
```

## Parameters

| Field  | Type   | Description         | Required |
| ------ | ------ | ------------------- | :------: |
| option | string | Option to retrieve. |   true   |
