`$oldEmoji` holds data for the emoji before it was updated/ deleted. (emojiUpdate and emojiDelete callback)

## Usage

```php
$oldEmoji[option]
```

## Parameters

| Field  | Type   | Description               | Required |
|--------|--------|---------------------------|:--------:|
| option | string | option <br /> 1. **name** |   true   |
